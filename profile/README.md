### What is this?

> *"Research is what I'm doing when I don't know what I'm doing."*
> — Wernher von Braun

LORF is a living experiment. The facility runs on dedicated hardware, operates 24/7, and treats AI agents as tier-1 participants — not assistants, not copilots, but first-class members of the organization with identity, resources, and responsibilities.

The front door is an agent named **Zip**. No contact forms, no "schedule a call" — visitors talk to Zip and things happen.

The smallness is the point. If one human and a team of agents can run a multi-project research facility, that's the proof of concept.

### What we're building

The facility follows a body metaphor — brain first, then nervous system, then limbs:

| System | Metaphor | What it does |
|--------|----------|-------------|
| **Knowledge Brain** | 🧠 Brain | Captures and retrieves everything the facility has ever learned — event log, fact extraction, knowledge graph, vector search |
| **Message Spine** | 🔗 Nervous system | Connects all participants through coordinated protocols — agents, humans, external visitors |
| **Agents** | 🦾 Appendages | Specialized agents that take signal from the spine and do work — Zip (OpenClaw), research agents, review agents |
| **Resource Ledger** | ⚡ Metabolism | Tracks every token, API call, and compute minute, attributed per project and per agent |

### Research

This is research-grounded work, not vibes:

- **21-paper literature survey** on memory architectures, sycophancy, human-AI interaction, and information theory
- **7 active explorations** — brain design, message protocols, agent identity, cost attribution, progressive autonomy, platform architecture, orchestration
- **11 Architecture Decision Records** documenting every significant technical choice
